## 1. 개요

### 1.1 목표 장면 (사용자가 정의한 경험)

출근 전에 앱을 켰을 때, 내가 아무것도 안 했는데 —

- 달팽이가 **버섯 뒤에서 자고** 있고
- **비가 와서 바위 밑으로 모여** 있고
- 두 마리가 **나란히 상추를 먹고**, 먹고 나서 **같이 쉬고** 있는

장면을 보게 한다. 게임이 아니라 **작은 디오라마**.

### 1.2 "AI"의 정의 (스택 제약 안에서)

LLM/서버가 아니라 **유틸리티 기반 행동 선택기(Behavior Director)** 다:
개체가 쉴 때마다 "지금 뭐 할까?"를 상황(날씨·시간대·성격·주변 장식·다른 달팽이)에 따라
가중치로 고르는 규칙 엔진. 결정적 요소(rng 주입)라 테스트 가능하고, 전부 연출 계층(HabitatModule)이다.
**게임 수치(성장/경제)에는 무영향** — 7차의 "성격은 연출만" 원칙을 생태계 전체로 확장한다.

### 1.3 요구사항 → 설계 매핑

| 요구 | 설계 |
|------|------|
| 장식 뒤에서 자기, 비 오면 바위 밑 | **앵커 행동**: 낮잠/휴식을 장식·가장자리 근처로 이동해서 한다. 비/밤엔 쉼터(바위·버섯) 선호 (§3) |
| 다른 달팽이랑 같이 먹고, 같이 쉬기 | **사회 행동**: 인사(더듬이 맞대기), 나란히 먹기 합류, 식후 같이 쉬기 (§4) |
| 아무것도 안 해도 살아있는 느낌 | 밤낮 리듬(밤엔 취침 + 어두운 오버레이), 이모트 말풍선(💤💕🎵)으로 행동을 읽히게 (§3.4, §3.5) |
| 부재 중에도 살아 있었다는 증거 | **부재 생활 시뮬레이션**: "달달이는 버섯 그늘에서 낮잠을 잤어요" 문장 생성 + **복귀 화면이 그 문장과 일치하는 장면으로 시작** (§5) |
| 유저 레벨로 최대 8마리 | 슬롯 확장을 양육자 레벨 게이트 + 코인 구매로 (§6) |

### 1.4 제외

- ❌ LLM/서버/실제 학습 — 규칙 기반 유틸리티 선택기만
- ❌ 교배/유전 — 여전히 범위 밖
- ❌ 사회 행동의 수치 보상 (행복 미세 보정 +1~2 수준의 연출성 소소함만 허용, 경제 무영향)
- ❌ 개체 간 갈등/싸움 — 힐링 톤 유지
- 이월 유지: 공유 카드/히든 변이/밤의 숲 맵 (9차 후보 §10)

## 2. 아키텍처 — 행동 선택기 (Behavior Director)

### 2.1 상태 확장

```
기존: idle / wandering / seeking_food / eating / napping
추가: resting(앵커 휴식) / socializing(다가가 인사) / watching(식사 구경)
```

### 2.2 선택 방식 (idle 만료 시)

각 후보 행동의 **기본 가중치 × 상황 배수**를 곱해 룰렛 선택:

| 행동 | 기본 | 상황 배수 (곱연산) |
|------|------|-------------------|
| 배회 (wander) | 40 | 개구쟁이 ×2 · 비 ×1.5 · 밤 ×0.3 |
| 낮잠 (nap→앵커) | 10 | 잠꾸러기 ×3 · 밤 ×5 · 비 ×1.3 |
| 앵커 휴식 (rest) | 20 | 장식 있음 ×1.5 · 배부름(hunger<20) ×1.5 |
| 어울리기 (socialize) | 15 | 동료 1마리↑ ×1.5(없으면 0) · 개구쟁이 ×1.5 |
| 식사 구경 (watch) | 0 | 먹는 동료 있으면 25 · 먹보 ×2 |

- 수치는 전부 `HabitatModule.BEHAVIOR` 상수 (연출 계층 — `GAME.CONFIG` 아님)
- rng 주입 가능 구조로 만들어 시드 테스트

### 2.3 앵커 (장소성)

- 앵커 = 배치된 장식 슬롯 위치 + 서식지 네 모서리 안쪽 지점
- 낮잠/휴식은 **가장 가까운 선호 앵커로 이동한 뒤** 수행 (이동하는 모습 자체가 디오라마)
- 선호도: 비·밤 = 이끼 바위 > 버섯 > 모서리 / 맑음 = 들꽃 > 조약돌 > 아무 곳
- 앵커에 도착하면 장식에 살짝 겹치게 앉음 (z-order: 장식 뒤 느낌은 앵커 오프셋으로 연출)

## 3. 생태 연출 (수치)

### 3.1 밤낮 리듬 (로컬 시간)

| 시간대 | 효과 |
|--------|------|
| 낮 (07~22시) | 기존 동작 |
| 밤 (22~07시) | 취침 시간 — 낮잠 가중 ×5, 수면은 앵커에서, 서식지에 남색 반투명 오버레이 + 🌙 |

- 밤에 클릭하면 깨어나 팝업 (깨우기 가능 — 페널티 없음)
- `body[data-daytime]`으로 CSS 제어, 판정은 `App`이 시간마다 갱신

### 3.2 날씨 연동 강화

- 비: 활동적(기존) + 낮잠·휴식 시 **쉼터(바위/버섯) 강제 선호** → "비 와서 돌 밑에 모임"
- 안개: 이동 느릿(×0.85), 휴식 가중 ×1.3

### 3.3 사회 행동 (§4 상세)의 빈도 가드

- 개체당 사회 행동 쿨다운 90초 (스팸 방지)
- 최대 동시 사회 상호작용 2쌍

### 3.4 이모트 말풍선

| 이모트 | 상황 |
|--------|------|
| 💤 | 잠들 때/자는 중 간헐 |
| 💕 | 인사 성공 순간 (양쪽) |
| 🎵 | 앵커 휴식 중 간헐 |
| 👀 | 식사 구경 시작 |
| ☔ | 비 오는 날 쉼터 도착 |

- 기존 `_floatAt` 재사용 + 잔류형 버블(.snail-emote, 2.5초) 추가

### 3.5 함께 먹기 (밸런스 무영향 설계)

- 배고픈 개체가 먹이를 점유하면, **다른 배고픈 개체도 같은 먹이에 합류 가능** (최대 2마리)
- 합류 개체도 먹는 모션 — 단, **정산은 각자 자기 몫**: 합류 시 재고에서 1개 더 소모될 상황이면 합류하지 않음(재고 2개 이상일 때만 같은 위치에서 각자 1개씩 소모로 처리)
- 즉 "한 접시에 두 마리" 그림을 만들되 경제는 기존 규칙(1마리 1먹이) 그대로

## 4. 사회 행동 시퀀스

```
socializing: 대상 선택(가장 가까운 idle/resting 동료) → 다가감 →
  인사: 마주보고 1.5초, 양쪽 💕 (행복 +1, 상한 무시 안 함)
  → 60% 확률로 "같이 쉬기": 둘이 근처 앵커로 이동해 나란히 resting
watching: 먹는 동료 옆 30px 지점으로 이동 → 👀 → 식사 끝나면
  → 50% 확률로 그 자리에서 같이 resting
```

- 대상이 이동/식사 시작하면 자연스럽게 취소 → idle
- 알은 사회 행동 대상 아님

## 5. 부재 중 생활 시뮬레이션 (살아 있었다는 증거)

### 5.1 생활 문장 생성 — `GAME.simulateAwayLife(snails, player, awayMinutes, dateKey, rng)`

- 순수 함수 (rng 주입). 부재 30분↑일 때 1~3개 문장 생성:
  - 소재: 날씨/시간대 + 성격 + 배치 장식 + 개체 조합
  - 예: "달달이는 버섯 그늘에서 오래 낮잠을 잤어요" / "비가 와서 다들 이끼 바위 밑에 모였어요" / "몽이와 달달이가 나란히 상추를 먹었어요" (실제 먹이 소모 없음 — 문장은 발견/감쇠와 별개의 '생활' 서사)
- 반환: `{ lines: [], scene: [{id, state: 'napping'|'resting', anchor: 'mushroom'|...}] }`
- 문장 중 대표 1개는 성장 일지에 기록 (하루 최대 2건 — 스팸 가드)

### 5.2 복귀 장면 배치 — 문장과 화면의 일치

- 부팅 시 `HabitatModule.applyScene(scene)`: 각 개체를 해당 앵커 근처 위치 + 해당 상태로 시작
- **리포트에서 읽은 문장이 홈 화면에서 실제로 보인다** — 이 일치가 "살아있음"의 증거
- 밤이면 취침 장면, 비면 쉼터 장면이 기본값

### 5.3 복귀 리포트 개편

```
🐌 다녀오셨어요? (7시간 12분)
─────────────────────────
🌙 달달이는 이끼 바위 옆에서 곤히 잤어요
💕 몽이와 달달이가 한참을 붙어 다녔어요
· 배고픔: 달달이 +25, 몽이 +25
· 돌아다니다 코인 8개를 주웠어요!
```

- 생활 문장(위) + 기존 스탯/발견(아래) 순서

## 9. 완료 조건

1. 아무 입력 없이 5분을 지켜보면: 배회·앵커 낮잠·앵커 휴식·(2마리↑) 인사/같이 쉬기가 자연스럽게 섞여 나온다
2. 비 오는 날 낮잠·휴식이 바위/버섯 근처로 모이고 ☔ 이모트가 뜬다
3. 밤(22~07시)에 열면 어두운 서식지에서 대부분 잠들어 있고, 클릭하면 깨어난다
4. 재고 2개 이상일 때 두 마리가 같은 자리에서 나란히 먹는 장면이 나온다 (소모는 각자 1개 — 경제 무영향)
5. 부재 30분↑ 복귀 시 생활 문장이 나오고, **홈 화면 첫 장면이 그 문장과 일치**한다
6. 생활 문장이 성장 일지에 하루 최대 2건 기록된다
7. 양육자 레벨이 오르면 슬롯이 8까지 열리고(§6 표), 미달 시 상점에 조건이 표시된다
8. 8마리 동시 배회가 실기기에서 버벅이지 않는다
9. 행동 선택기가 rng 주입으로 결정적으로 테스트된다 + 통합 테스트 전면 갱신 통과 + 콘솔 에러 0
10. 기존 기능(성장/경제/여행/탐험/백업/PWA) 무회귀 — 생태계는 전부 연출 계층
