> 선행 조건: 6차 MVP + restyle 완료 (v0.8.0)
> 목표 기간: 3~4주
> 검증 가설: **"영속 성장(양육자 레벨)과 개체 중심 UI(팝업)가 돌봄을 더 깊게 만드는가?"**

---

## 1. 개요

### 1.1 요구사항 → 설계 매핑

| 요구 | 설계 |
|------|------|
| 유저 레벨 | **양육자 레벨** — 모든 돌봄 행동으로 XP. 달팽이는 여행을 떠나도 양육자 레벨은 영속. 먹이 해금·탐험 스태미나의 게이트 (§3) |
| 성격은 단순하게 (스탯 변화 없음) | 성격 = **행동 연출만**. 잠꾸러기는 잠을 유난히 많이 자고, 개구쟁이는 유난히 활발하게 돌아다닌다. 게임 수치에는 무영향 (§4) |
| 장식 기능화 | 배치한 장식마다 패시브 효과 (§5) |
| 먹이 다양화 (딸기 → 사과) | 먹이 4종: 상추/당근/**사과**/특제 샐러드 — 가격·회복·경험치 상이, 양육자 레벨로 해금 (§6) |
| 달팽이 탭 → 유저 탭 | 탭을 **계정 단위 화면**으로 개편: 양육자 카드 + 도감 + 앨범 + 일지 (§7.1) |
| 달팽이 축소 + 머리 위 수치 | 스프라이트 ~30% 축소, 머리 위에 배고픔/행복 미니 수치 상시 표시 (§7.2) |
| 달팽이 클릭 → 상세 창 | 개체 팝업: 스탯/성격/변이 + [먹이주기]/[쓰다듬기]/[여행 보내기] (§7.3). 개체 상세는 탭에서 팝업으로 이동 |

### 1.2 시스템 연결도

```
         ┌─ 모든 돌봄 행동 ─────────────┐
         ▼                              │
   양육자 레벨(XP) ──해금──▶ 먹이 4종 ──▶ 달팽이 성장 (레벨/여행)
         │                                     │
         └─해금──▶ 탐험 스태미나                 ▼
                                        도감/앨범/세대 (유저 탭)
   장식(배치) ─수정치─▶ 감쇠/회복 계산
   성격 ─행동 연출만─▶ 서식지 움직임 (수치 무영향)
```

### 1.3 제외

- ❌ 성격의 게임 수치 효과 (경험치/감쇠/확률 보정 없음 — 행동 연출만)
- ❌ 유저 레벨 소셜 요소(랭킹) — 서버 없음
- ❌ 먹이 전용 일러스트 — 미니 인라인 SVG로 (캐릭터 아트는 사용자 담당)
- ❌ 성격 추가/변경 아이템, 장식 신규 콘텐츠
- 이전 제안(공유 카드/히든 변이/신규 맵/PWA 알림)은 8차 이월 (§10)

## 3. 양육자 레벨 (수치)

### 3.1 XP 획득 (계정 단위)

| 행동 | XP | 비고 |
|------|-----|------|
| 먹이 주기 | +2 | 먹이 경제가 자연 상한 |
| 탐험 뒤지기 | +1 | 스태미나가 상한 |
| 데일리 접속 | +5 | |
| 미션 1개 달성 | +5 | |
| 미션 완주 보너스 | +10 | |
| 부화 | +15 | |
| 여행 보내기 | +30 | |
| 도감 신규 발견 | +25 | |
| 쓰다듬기 | 0 | 쿨다운이 없어 XP 파밍 방지 — 미션 XP로만 반영 |

### 3.2 레벨 곡선과 보상

- 필요 XP = `50 + (level − 1) × 25` (Lv2: 50, Lv3: 75, Lv4: 100 …)
- 일일 예상 XP ≈ 55 → **Lv2 첫날, Lv4 ~4일차, Lv6 ~10일차**
- 레벨업 보상: 코인 `30 × 새 레벨` + 팡파레/컨페티 + 일지 기록

### 3.3 해금 테이블

| 양육자 Lv | 해금 |
|-----------|------|
| 2 | 먹이 "당근" |
| 4 | 먹이 "사과" |
| 5 | 탐험 스태미나 10 → 12 |
| 6 | 먹이 "특제 샐러드" |
| 8 | 탐험 스태미나 12 → 14 |

## 4. 성격 — 행동 연출만 (게임 수치 무영향)

| 성격 | id (저장값 유지) | 행동 연출 |
|------|-----------------|-----------|
| 먹보 🍽️ | `foodie` | 먹이를 눈에 띄게 빨리 쫓아감 (추적 속도 1.4배), 먹는 시간 짧음 (0.7배) |
| 개구쟁이 🤸 | `explorer` (라벨만 개명) | 가만히 못 있음 — idle 시간 0.5배, 이동 속도 1.25배 |
| 잠꾸러기 😴 | `sleepy` | 잠이 정말 많음 — 낮잠 확률 3배, 낮잠 시간 1.5배, 이동 느릿(0.85배) |

- 전부 `HabitatModule.MOTION` 배수 (연출 계층) — `GAME` 수치에는 손대지 않는다
- 팝업/유저 탭 표기 문구: "개구쟁이 — 잠시도 가만히 있지 못해요" 식의 관찰 문구

## 5. 장식 효과 (패시브)

| 장식 | 효과 |
|------|------|
| 조약돌 | 행복 감소 15% 완화 |
| 버섯 | 먹이의 배고픔 회복 +10% |
| 들꽃 | 쓰다듬기 행복 +3 추가 |
| 이끼 바위 | 배고픔 증가 10% 완화 |

- **배치된 슬롯의 장식만 발동** (슬롯 3 < 장식 4 → 선택)
- `GAME.decorationEffects(player)` → `{ hungerDecayMult, happinessDecayMult, feedHungerMult, petHappinessBonus }`
- `applyTimeDecay(..., mods)` 시그니처 확장 (생략 시 기존 동작), 상점에 효과 표기

## 6. 먹이 4종 (수치)

| 먹이 | 가격 | 배고픔 | 경험치 | 행복 | 해금 | 용도 |
|------|------|--------|--------|------|------|------|
| 🥬 상추 | 10 | −30 | +10 | +5 | 기본 | 기본기 |
| 🥕 당근 | 18 | −45 | +14 | +5 | 양육자 Lv2 | 가성비 포만 |
| 🍎 사과 | 30 | −35 | +16 | +12 | 양육자 Lv4 | 행복 케어 |
| 🥗 특제 샐러드 | 60 | 완전 회복 | +30 | +15 | 양육자 Lv6 | 프리미엄 |

- ×10 묶음 10% 할인 (전 먹이 공통), 부재 발견/탐험 습득은 상추 고정
- `player.foods` 인벤토리 + `selected_food` (schema v6, 기존 `food` 숫자 → `foods.lettuce`)
- [먹이] FAB 탭 = 선택 먹이 드롭, FAB 길게/시트로 종류 전환 (보유·잠금 표시). 지갑 칩은 선택 먹이 아이콘+수량
- 먹이 비주얼: 미니 인라인 SVG 3종 추가 (당근/사과/샐러드)
- `GAME.feed(snail, player, foodId)` 확장, `FOOD_DEFS` 테이블이 단일 기준

## 7. UI 개편

### 7.1 유저 탭 (구 달팽이 탭)

```
[유저 탭]  (탭 아이콘: 👤, 라벨 "유저")
├─ 🧑‍🌾 양육자 카드 — Lv/XP 바/다음 해금 안내
├─ 📖 도감 (변이 수집)
├─ 🏆 앨범 (역대 달팽이)
└─ 📜 성장 일지
```

- 개체 스탯/특성/여행 UI는 **팝업으로 이동** — 탭은 계정 단위 정보 전용
- 개체 선택기 제거 (더 이상 불필요)

### 7.2 서식지: 축소 + 머리 위 수치

- 스프라이트 크기 ~30% 축소: 아기 44 / 어린 64 / 성체 84px — 여러 마리+수치 표시 공간 확보
- 각 개체 머리 위에 **미니 스탯 배지** 상시 표시: `🍀65 😊80` (배고픔/행복 수치, 반투명 필)
- 배지는 개체 이동을 따라다니고, 스탯 변화 시 즉시 갱신. 알은 배지 없음

### 7.3 개체 팝업 (달팽이 클릭)

```
┌───────────────────────────┐
│  [달팽이 일러스트]  달달이    │
│  Lv.5 어린 · 올리브 · 먹보   │
│  "먹이를 보면 눈빛이 변해요"  │
│  ⭐ 경험치  ▓▓▓░░  32/100   │
│  🍀 배고픔  ▓▓░░░  40%      │
│  😊 행복    ▓▓▓▓░  85%      │
│  [🥬 먹이주기] [🖐️ 쓰다듬기] │
│  [🧳 여행 보내기]※조건 충족 시│
│            [닫기]           │
└───────────────────────────┘
```

- **달팽이 클릭 = 팝업 열림** (기존 "터치 = 쓰다듬기"를 대체 — 쓰다듬기는 팝업 버튼으로)
- 먹이주기: 선택된 먹이를 **그 달팽이 근처에 드롭** (그 아이가 먹도록)
- 쓰다듬기: 기존 효과 + 하트, 미션 반영. 행동 후 팝업 수치 즉시 갱신
- 여행 보내기: 조건(성체 Lv12+) 충족 시 노출, 기존 확인 플로우
- 빈 곳 터치 = 먹이 드롭은 유지

## 10. 완료 조건

1. 돌봄 행동으로 양육자 XP가 쌓이고 레벨업 보상·해금(§3.3)이 동작한다
2. 먹이 4종(사과 포함)이 각기 다른 효과로 동작하고 선택 먹이가 드롭·지갑에 반영된다
3. 달팽이가 축소된 크기로 머리 위 배고픔/행복 수치를 달고 다닌다
4. 달팽이 클릭 시 팝업이 열리고 먹이주기/쓰다듬기/여행이 팝업에서 동작한다
5. 유저 탭에 양육자/도감/앨범/일지만 남고 개체 상세는 팝업으로 완전 이관된다
6. 잠꾸러기는 눈에 띄게 잠이 많고 개구쟁이는 눈에 띄게 활발하다 (게임 수치는 3성격 모두 동일)
7. 장식 배치/해제에 따라 패시브가 켜지고 꺼진다
8. v5 데이터 무손실 마이그레이션 (상추 수 이전)
9. 통합 테스트 전면 갱신 통과 + 콘솔 에러 0개
10. 기존 기능(멀티/탐험/여행/백업/PWA) 무회귀
