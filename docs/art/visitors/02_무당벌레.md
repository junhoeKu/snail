# 방문객: 무당벌레 (visitor_ladybug — 14차 Phase 4 동적 이벤트)

| 항목 | 값 |
|------|-----|
| 용도 | 서식지 바닥을 가끔 기어가는 **연출 전용 방문객** (게임 수치 무영향) |
| 파일명 | 원본 `docs/art/visitors/무당벌레.png` → 배포 `assets/visitors/visitor_ladybug.png` (배치·후처리는 구현 단계에서) |
| 규격 | **1024×1024 PNG, 투명 배경, 왼쪽을 향한 옆모습** (좌우 반전 로직 `FLIP_RIGHT: -1` 공용) |
| 수량 | 1장 (기어가는 모션은 CSS 애니메이션으로 처리) |

> ⚠️ **에픽 변이 "무당벌레 달팽이"(snail_ladybug)와 다른 존재다** — 이것은 달팽이가 아닌
> 진짜 곤충 방문객. 혼동을 피하려 껍질 무늬가 아니라 곤충의 둥근 딱지날개로 그린다.
> 이 폴더(docs/art/visitors)는 release.py의 characters 글롭과 분리 — manifest.json에 등록하지 않는다.

## 구도 제약

- 옆모습, **왼쪽으로 기어가는 방향** — 다리가 살짝 보이는 낮은 자세
- 몸통이 캔버스 폭의 ~65% (화면에서 24~32px로 작게 쓰임 — 실루엣이 단순해야 함)
- **차분한 적갈색 딱지날개 + 검은 점 4~6개** — 달팽이 레드 변이보다 채도 낮게
- 그림자·배경 요소 없음 (STYLE_GUIDE §2 동일)

## 결과 확인 체크리스트

- [ ] 왼쪽을 향해 기는 방향인가 (반전 로직 기준)
- [ ] 28px로 축소해도 무당벌레로 읽히는가
- [ ] 에픽 무당벌레 달팽이와 한 화면에 있어도 헷갈리지 않는가
- [ ] 투명 배경에 잔여 픽셀이 없는가
