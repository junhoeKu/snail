# 방문객: 나비 (visitor_butterfly — 14차 Phase 4 동적 이벤트)

| 항목 | 값 |
|------|-----|
| 용도 | 서식지를 가끔 가로질러 날아가는 **연출 전용 방문객** (게임 수치 무영향) |
| 파일명 | 원본 `docs/art/visitors/나비.png` → 배포 `assets/visitors/visitor_butterfly.png` (배치·후처리는 구현 단계에서) |
| 규격 | **1024×1024 PNG, 투명 배경, 왼쪽을 향한 옆모습** (좌우 반전 로직 `FLIP_RIGHT: -1` 공용) |
| 수량 | 1장 (날갯짓은 CSS 애니메이션으로 처리 — 프레임 분할 불필요) |

> ⚠️ 이 폴더(docs/art/visitors)는 달팽이 스프라이트 파이프라인(release.py의
> docs/art/characters 글롭)과 **분리**되어 있다. manifest.json에 등록하지 않는다.

## 구도 제약

- 옆모습, **왼쪽으로 나는 방향** — 날개는 위로 살짝 접힌 순간(∧자)이 CSS 펄럭임과 잘 맞는다
- 몸통+날개가 캔버스 폭의 ~70% (달팽이보다 여백을 조금 더 — 화면에서 32~40px로 작게 쓰임)
- 달팽이 팔레트와 겹치지 않는 **부드러운 미색/연노랑 날개** (배경 초록 위에서 잘 보이되 튀지 않게)
- 그림자·배경 요소 없음 (STYLE_GUIDE §2 동일)

```
no background elements, no text, no watermark
```
