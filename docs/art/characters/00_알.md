# 캐릭터: 알 (egg)

| 항목 | 값 |
|------|-----|
| 인게임 id | `stage: egg` |
| 용도 | 온보딩(이름 짓기) 화면 |
| 파일명 | `assets/characters/egg.png` |
| 규격 | 1024×1024 PNG, 투명 배경 |

모든 세대가 같은 알에서 시작하므로 1장만 있으면 된다. 흔들리는 연출은 CSS(`egg-wobble`)가 담당하니 정적인 그림 1장이면 충분하다.
