# 캐릭터: 황금 달팽이 (golden) — 희귀 변이

| 항목 | 값 |
|------|-----|
| 인게임 id | `color: golden` |
| 등장 확률 | **2%** (1세대 기준 — 최고 희귀) |
| 파일명 | `snail_golden_baby.png` · `snail_golden_junior.png` · `snail_golden_adult.png` |
| 규격 | 1024×1024 PNG, 투명 배경, 왼쪽을 향한 옆모습 |

> 갈색 성체 이미지를 **참조로 첨부**하고 생성한다.
> 분위기 키워드: 특별함, 은은한 빛, 행운. **과하게 번쩍이면 안 된다** — 동화책 톤 안에서의 금빛.
> 인게임 반짝임 애니메이션(`golden-glow`)은 CSS가 담당하므로 그림 자체의 발광은 절제할 것.
