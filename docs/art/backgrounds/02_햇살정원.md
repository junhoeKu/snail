# 배경: 햇살 정원 (garden)

| 항목 | 값 |
|------|-----|
| 인게임 id | `background: garden` |
| 파일명 | `assets/backgrounds/bg_garden.png` |

| 날씨 변형 | 불필요 — 비/안개는 CSS 오버레이 담당, **맑은 기본 장면 1장**만 |

## 구도 제약 (이끼 숲과 동일 — 중요)

- 하단 2/3는 트인 흙/모래 바닥 (달팽이 무대)
- 디테일(화분·들꽃·조약돌)은 좌우 가장자리와 상단에만
- 네 모서리 차분하게 (UI 칩 자리), 중간 명도 유지
- 이끼 숲과 **같은 그림체** — 가능하면 이끼 숲 결과물을 참조 이미지로 첨부
