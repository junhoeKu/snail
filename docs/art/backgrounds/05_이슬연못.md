> **햇살정원(garden) 은퇴 계획**: 배경 라인업을 이끼숲 리디자인(default) + 이슬연못(pond) +
> 고사리계곡(fern) 3종으로 교체한다. 구현 시 함께 바꿀 것 —
> ① `css/theme.css`의 `body[data-background]` 규칙 교체 ② 배경 선택 UI(index.html `data-bg` + deco.js)
> ③ 서버 검증 `backend/app/modules/game.py`의 `("default", "garden")` 튜플
> ④ **구버전 사용자의 저장값 `background: 'garden'` 폴백**(default로 치유 — 클라 self-heal + 서버 검증)
> ⑤ `assets/backgrounds/bg_garden.jpg` 제거는 폴백 배포 후(캐시된 구클라 대비 1릴리스 유예).
> 탐험 맵 "이슬 연못"과 같은 세계관 장소다 — 탐험에서 본 곳에서 살게 되는 연결감을 의도.

## 구도 제약 (게임 UI와의 합성 조건 — 중요)

- **하단 2/3는 달팽이가 돌아다니는 무대** — 물이 아니라 **연못가의 촉촉한 이끼·풀 바닥**으로 비워둘 것 (물 위를 기는 그림 금지)
- **연못 수면은 상단 1/3에만** — 왼쪽 위나 오른쪽 위로 치우쳐 살짝 보이는 물가, 은은한 반사광
- 이슬 맺힌 풀잎·물풀·둥근 돌은 **좌우 가장자리에만**
- **네 모서리는 차분하게** — UI 칩 4개가 올라감 (특히 수면 반짝임이 우상단 지갑 칩과 겹치지 않게)
- 전체적으로 **중간 명도**, 청록이 섞인 초록 팔레트 — 기존 이끼숲(#74905f)보다 반 톤 시원하게, 채도는 낮게
- 달팽이 스프라이트(갈색~노랑 계열)와 채도 경쟁 금지
