> 햇살정원(garden) 은퇴 계획과 교체 절차는 [05_이슬연못.md](05_이슬연못.md) 상단 노트를 따른다.
> 콘셉트: 이끼숲(아늑한 기본) · 이슬연못(시원한 물가)과 구분되는 **깊고 그늘진 초록 계곡** —
> 양치식물 사이로 내려오는 빛줄기가 포인트. 탐험 맵 세계관의 "더 깊은 숲" 방향.

## 구도 제약 (게임 UI와의 합성 조건 — 중요)

- **하단 2/3는 달팽이가 돌아다니는 무대** — 부드러운 흙+이끼 바닥으로 비워둘 것 (계곡물·바위 지형으로 채우지 않기)
- **고사리 잎은 좌우 가장자리와 상단에만** — 잎이 크므로 화면 안쪽으로 1/4 이상 침범 금지
- 상단 1/3은 겹겹의 고사리·수풀 실루엣으로 **깊이감** (03번 리디자인과 같은 원경 문법)
- 빛줄기(god ray)는 **은은하게 1~2개** — 위에서 아래로, UI 칩·무대 중앙을 강하게 비추지 않게
- **네 모서리는 차분하게** — UI 칩 4개 자리
- 그늘진 장소지만 **중간 명도 유지** — 너무 어두우면 흰색 칩·달팽이 그림자와 싸운다. 깊은 초록+이끼 초록, 채도 낮게
