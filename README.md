# 🐌 Snail — 달팽이 키우기

달팽이 한 마리를 부화시켜 먹이를 주고 성장시키는 **데일리 케어 게임**입니다.
현실 시간이 흐르면 달팽이가 배고파지고(1시간마다 +5), 다시 돌아와 돌봐줄수록 성장합니다.

```
알 획득 → 부화 → 먹이 주기 → 성장 → 코인 획득 → 먹이 구매 → 더 성장
```

## 주요 기능

- 🥚 **부화**: 알에 이름을 지어주면 달팽이가 태어납니다
- 🌿 **서식지**: 화면 전체가 달팽이의 서식지입니다. 달팽이가 스스로 돌아다니고, 아무 곳이나 터치하면 그 자리에 상추가 떨어져 달팽이가 다가가 먹습니다
- 📱 **하단 5탭**: 스탯 · 상점 · 홈 · 꾸미기 · 설정 — 중앙 홈 버튼으로 언제든 서식지로
- 🎨 **실사풍 비주얼**: SVG 달팽이와 자연 톤 텍스처 배경 (이끼 숲 / 햇살 정원)
- 🥬 **먹이 주기**: 상추를 주면 배고픔이 줄고 경험치를 얻습니다
- 📈 **성장**: 레벨업 → Lv5 껍질이 커지고, Lv10 색이 변합니다
- 🪙 **코인**: 먹이 주기 / 산책(4시간 쿨다운) / 일일 접속 보상으로 획득
- ⏰ **시간 시스템**: 미접속 시간도 정산되고, 돌아오면 "그동안 있었던 일" 리포트를 보여줍니다 (가끔 달팽이가 뭔가 주워오기도!)
- 📋 **데일리 루프**: 출석 스트릭(연속 보너스) + 오늘의 돌봄 미션 3종(먹이/산책/쓰다듬기)
- 🖐️ **쓰다듬기**: 달팽이를 직접 터치하면 좋아해요 (30분마다)
- 🌦️ **살아있는 하루**: 날씨(비 오는 날엔 신남), 낮잠, 성격 3종(먹보/모험가/잠꾸러기), 배고프면 시무룩
- 🐚 **껍질 변이**: 부화 시 5가지 껍질 색 중 하나 — 2% 확률의 황금 달팽이도!
- 📖 **성장 일지**: 함께한 순간들이 타임라인으로 쌓입니다
- 💾 **저장**: LocalStorage에 자동 저장 — 껐다 켜도 그대로 이어집니다

## 실행 방법

빌드 과정이 없습니다. 정적 서버로 열기만 하면 됩니다.

```bash
python3 -m http.server 31111
# → http://localhost:31111
```

또는 VS Code Live Server 확장을 사용하세요.
