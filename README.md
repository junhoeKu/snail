# 🐌 Snail — 달팽이 키우기

> **▶️ 바로 플레이: https://junhoeku.github.io/snail/**

달팽이 한 마리를 부화시켜 먹이를 주고 성장시키는 **데일리 케어 게임**입니다.
현실 시간이 흐르면 달팽이가 배고파지고(1시간마다 +5), 다시 돌아와 돌봐줄수록 성장합니다.

```
알 획득 → 부화 → 먹이 주기 → 성장 → 코인 획득 → 먹이 구매 → 더 성장
```

## 주요 기능

- 🥚 **부화**: 알에 이름을 지어주면 달팽이가 태어납니다
- 🐌🐌🐌 **멀티 사육**: 상점에서 보금자리를 넓혀 최대 3마리를 동시에 키워요 — 각자 성격도 성장도 다릅니다
- 🌿 **서식지**: 화면 전체가 서식지입니다. 아무 곳이나 터치하면 상추가 떨어지고(여러 개 가능!), 배고픈 아이들이 가까운 상추부터 나눠 먹어요
- 🧭 **탐험 채집**: 맵에 들어가 수풀을 뒤지면 코인·상추, 운 좋으면 야생 알까지 — 맵마다 잘 나오는 변이가 달라요
- 📱 **하단 5탭**: 스탯 · 상점 · 홈 · 꾸미기 · 설정 — 중앙 홈 버튼으로 언제든 서식지로
- 🎨 **실사풍 비주얼**: SVG 달팽이와 자연 톤 텍스처 배경 (이끼 숲 / 햇살 정원)
- 🥬 **먹이 주기**: 상추를 주면 배고픔이 줄고 경험치를 얻습니다
- 📈 **성장**: 레벨업 → Lv5 껍질이 커지고, Lv10 색이 변합니다
- 🪙 **코인**: 먹이 주기 / 일일 접속 보상 / 달팽이가 주워오는 발견으로 획득
- ⏰ **시간 시스템**: 미접속 시간도 정산되고, 돌아오면 "그동안 있었던 일" 리포트를 보여줍니다 (가끔 달팽이가 뭔가 주워오기도!)
- 📋 **데일리 루프**: 출석 스트릭(연속 보너스) + 오늘의 돌봄 미션 3종(먹이/쓰다듬기/탐험)
- 📱 **앱 설치(PWA)**: 홈 화면에 설치하면 오프라인에서도 열리고 데이터가 더 안전해요
- 💾 **백업 코드**: 설정에서 코드 하나로 내보내고, 다른 기기에서 그대로 복구
- 🖐️ **쓰다듬기**: 달팽이를 직접 터치하면 언제든 좋아해요
- 🌦️ **살아있는 하루**: 날씨(비 오는 날엔 신남), 낮잠, 성격 3종(먹보/모험가/잠꾸러기), 배고프면 시무룩
- 🐚 **껍질 변이 & 도감**: 부화 시 5가지 껍질 색 중 하나 — 2% 확률의 황금 달팽이도! 도감을 채워보세요
- 🧳 **여행 보내기**: 성체 Lv.12가 되면 여행을 보내고 새 알을 맞이해요. 세대가 오를수록 희귀 변이 확률 상승, 떠난 달팽이는 앨범에
- 🪴 **장식**: 조약돌·버섯·들꽃·이끼 바위를 구매/해금해 서식지를 꾸며요
- 🔊 **게임필**: 합성 효과음(설정에서 끄기 가능), 코인 플라이, 컨페티
- 📖 **성장 일지**: 함께한 순간들이 타임라인으로 쌓입니다
- 💾 **저장**: LocalStorage에 자동 저장 — 껐다 켜도 그대로 이어집니다

## 실행 방법

빌드 과정이 없습니다. 정적 서버로 열기만 하면 됩니다.

```bash
python3 -m http.server 31111
# → http://localhost:31111
```

또는 VS Code Live Server 확장을 사용하세요.
